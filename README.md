# Advanced Excel Assignments and Activities

This repository is dedicated to storing all **Advanced Excel assignments and activities** completed as part of coursework or self-learning. It will include practice files, assignments, and exercises to demonstrate advanced Excel skills.

## Purpose

The purpose of this repository is to:

- Track and organize all advanced Excel work in one place.
- Showcase proficiency in Excel functions, formulas, and data analysis techniques.
- Serve as a reference for future Excel projects or learning.

## Skills Demonstrated

- Advanced formulas (VLOOKUP, INDEX-MATCH, IF, nested functions, etc.)
- Pivot tables and charts
- Data analysis and visualization
- Conditional formatting and data validation
- Automation with Excel features
